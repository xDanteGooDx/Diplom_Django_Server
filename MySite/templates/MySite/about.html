О проекте

Тема проекта: "Создание мобильного портала учебной литературы для IT-специальностей"

Проект разработан Коньюковым Михаилом, студентом Московского Авиационного Института(МАИ), группы 8-3ИНТ-4ДБ-035-15, под научным руководством Чумаковой Екатерины Витальевны, доцента, кандидата физико-математических наук

Проект предоставляет людям, которые хотят изучить "мир" программирования, узко специализированную библиотеку по IT-сфере, а так же позволяет преподавателям, программистам, специалистам в IT-сфере выкладывать свои материалы для обучения нового поколения программистов, специалистов в IT-сфере

Сервер написан на Django framework, использует также Django-rest-framework, для обработки http-запросов и выдачи JSON-ответов. Django-rest-framework используется для поддержки Desktop-клиентов. Используется также встроенный в Django: Django - ORM — технология программирования, которая связывает базы данных с концепциями объектно-ориентированных языков программирования, создавая «виртуальную объектную базу данных»
